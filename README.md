# 🌌 太陽系シミュレーター

プロシージャル描画によるインタラクティブな太陽系・銀河系・惑星表面の3Dシミュレーター。外部画像アセット不使用、React + Canvas 2D APIのみで実装。

## 🚀 ライブデモ

[https://htanny.github.io/solar/](https://htanny.github.io/solar/)

## 📦 セットアップ

```bash
git clone https://github.com/htanny/solar.git
cd solar
npm install
npm run dev      # 開発サーバー http://localhost:5173
npm run build    # 本番ビルド
```

---

## 概要

太陽系の8惑星・太陽・月・追加衛星・ガリレオ衛星・彗星・小惑星帯・矮小惑星・探査機・ラグランジュ点・CME、天の川銀河の3D構造、全8惑星の表面風景を、プロシージャルに描画するシミュレーター。

- **外部画像なし** — Canvas 2Dによる完全プロシージャル描画
- **物理計算** — ケプラー軌道、自転、地軸傾斜、太陽日、位相角影
- **3階層スケール** — 太陽系 / 恒星間 / 銀河系
- **着陸ビュー** — 各惑星の地表から周囲を見渡す一人称視点
- **時間旅行** — 任意の日付にジャンプして惑星配置を再現
- **ツアーモード** — 太陽→全惑星を自動巡回

### ビジュアル

- 全惑星プロシージャルテクスチャ（地球の7大陸、木星の大赤斑、土星リング等）
- 位相角ベースの昼夜影（月の満ち欠けと同じ原理）
- 背景1400個の恒星（OBAF分類の8色、まばたき効果）
- 10個の星雲
- 軌道線・軌跡（フェード）・地軸表示
- 距離比較バー（対数スケール、全惑星を一列に表示）
- 2天体間距離計測ツール（AU・光到達時間）

### モード

- **太陽系ビュー** — 標準モード
- **銀河系ビュー** — ズームアウトで天の川銀河を3D回転視点で観察
- **惑星表面ビュー** — 各惑星の地表から周囲を見渡す（着陸モード）
- **サイズ比較モード** — 太陽・全惑星を実スケールで並列表示
- **ツアーモード** — 太陽から海王星まで自動巡回

#### HUD（着陸モード）

- **流星群表示**（放射点マーク・流星レート付き）
- FOV倍率（望遠鏡使用時）

## UIパネル

全パネルは**ドラッグで移動可能**（⠿マーク）

### 左上: フォーカスパネル
全体・太陽・各惑星・彗星から選択

### 右上: 速度パネル
⏸（一時停止）・0.5x・1x・4x・15x・50x・100x

### 左下: 表示パネル

- 軌道・軌跡・小惑星帯・地軸・月・ラベル・惑星 ON/OFF
- **追加トグル**: L点（ラグランジュ）・探査機・準惑星名・CME・距離バー
- 実スケール（太陽・惑星・距離）
- 統一比率・**比較モード**・**ツアー**・BGM・EN/JA
- **📐計測**: 2天体間の距離を計測
- ツール: 日付移動・📷撮影モード・🔗共有・📥読込・今日

### 右下: ズームパネル
`+` / `−` ボタンと現在ズーム表示（太陽系 / 恒星間 / 銀河）

### 情報パネル（天体選択時）
- 質量・半径・重力・自転/公転周期・衛星数・大気・気温・地軸傾斜
- **地球からの現在距離・光到達時間・視直径・等級**
- 水星/金星: 位相（%・位相角）
- **🚀着陸ボタン**（惑星のみ、ツアー中は無効）

## 物理モデル

### 軌道力学
- 惑星: 円軌道近似
- 彗星・命名小惑星・探査機: **ケプラー方程式** `E = M + e·sin(E)` をニュートン法で解法
- 真近点角: `v = 2·atan2(√(1+e)·sin(E/2), √(1-e)·cos(E/2))`

### 自転
- 太陽日計算: 順行時 `1/(1/rotAbs − 1/p)`、逆行時 `1/(1/rotAbs + 1/p)`
- 金星（243日逆行）→ 太陽日117日で西から昇り東へ
- 天王星（97.8°傾斜）→ 地軸が公転面にほぼ水平

### 影（昼夜）
- 3D空間で太陽方向ベクトルを計算し位相角に基づいてターミネーターを楕円描画

### 太陽高度（着陸ビュー）
```
sin(高度) = sin(緯度)·sin(赤緯) + cos(緯度)·cos(赤緯)·cos(時角)
```

### 地球からの見かけ等級
```
m ≈ H + 5·log₁₀(r_helio · r_geo)
```

## トラブルシューティング

### 📷 撮影モードで画像が保存できない
Artifact環境では `canvas.toDataURL` がブロックされます。撮影モードはUIパネル非表示状態にするだけです。OSのスクリーンショット機能を使用してください。

### 🔗 共有URLが機能しない
状態コード方式を使用してください：
1. 「🔗 共有」で状態コードを取得（例: `SS|9613.2|-0.500|0.300|17|Earth`）
2. 「📥 読込」にコードを貼り付けて「復元」

### BGM が鳴らない
ブラウザのオートプレイ制限のため、BGMボタンをタップして明示的に有効化してください。

### 動作が重い
- 軌跡・小惑星帯・ラベルをOFFにする
- CME・探査機・L点をOFFにする
- 統一比率モードでの惑星フォーカスは高負荷
