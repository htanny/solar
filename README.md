# 🌌 太陽系シミュレーター

プロシージャル描画によるインタラクティブな太陽系・銀河系・惑星表面の3Dシミュレーター。外部画像アセット不使用、React + Canvas 2D APIのみで実装。

## 🚀 ライブデモ

[https://htanny.github.io/solar/](https://htanny.github.io/solar/)

## 📦 セットアップ

```bash
git clone https://github.com/htanny/solar.git
cd solar
npm install
npm run dev      # 開発サーバー http://localhost:5173
npm run build    # 本番ビルド
```

---

## 概要

太陽系の8惑星・太陽・月・追加衛星・ガリレオ衛星・彗星・小惑星帯・矮小惑星・探査機・ラグランジュ点・CME、天の川銀河の3D構造、20天体以上の表面風景を、プロシージャルに描画するシミュレーター。

- **外部画像なし** — Canvas 2Dによる完全プロシージャル描画
- **物理計算** — ケプラー軌道、自転、地軸傾斜、太陽日、位相角影
- **3階層スケール** — 太陽系 / 恒星間 / 銀河系
- **着陸ビュー** — 20天体以上の地表から周囲を見渡す一人称視点
- **時間旅行** — 任意の日付にジャンプして惑星配置を再現
- **ツアーモード** — 太陽→全惑星を自動巡回

## 物理モデル

### 軌道力学
- 惑星・準惑星: **楕円軌道**（離心率 e・近日点黄経 ϖ を実データで保持、太陽を焦点に配置）
- 彗星・命名小惑星・探査機: **ケプラー方程式** `E = M + e·sin(E)` をニュートン法で解法
- 真近点角: `v = 2·atan2(√(1+e)·sin(E/2), √(1-e)·cos(E/2))`
- 動径: `r = a(1−e²)/(1+e·cos v)`（水星 e=0.206、冥王星 e=0.249 など離心が視認可能）

### 自転
- 太陽日計算: 順行時 `1/(1/rotAbs − 1/p)`、逆行時 `1/(1/rotAbs + 1/p)`
- 金星（243日逆行）→ 太陽日117日で西から昇り東へ
- 天王星（97.8°傾斜）→ 地軸が公転面にほぼ水平

### 影（昼夜）
- 3D空間で太陽方向ベクトルを計算し位相角に基づいてターミネーターを楕円描画

### 太陽高度（着陸ビュー）
```
sin(高度) = sin(緯度)·sin(赤緯) + cos(緯度)·cos(赤緯)·cos(時角)
```

### 地球からの見かけ等級
```
m ≈ H + 5·log₁₀(r_helio · r_geo)
```

## 開発

### コマンド

```bash
npm run dev            # 開発サーバー起動 (http://localhost:5173)
npm run build          # 本番ビルド
npm run preview        # ビルド結果をプレビュー
npm run test:unit      # Vitest ユニットテスト実行
npm run test:unit:watch # Vitest ウォッチモード
npm run typecheck      # JSDoc + @ts-check 静的型チェック
npm test               # Playwright ビジュアル回帰テスト
```

### 型チェック方針

- `tsconfig.json` で `checkJs:false`（オプトイン方式）
- ファイル先頭に `// @ts-check` を付けたファイルのみ型検査対象（`drawLanding*.js` 系3ファイル）
- `@typedef` を `drawLanding.js` に集中定義し、子ファイルは `import("./drawLanding.js").TypeName` 形式で参照

## トラブルシューティング

### 📷 撮影モードで画像が保存できない
Artifact環境では `canvas.toDataURL` がブロックされます。撮影モードはUIパネル非表示状態にするだけです。OSのスクリーンショット機能を使用してください。

### 🔗 共有URLが機能しない
状態コード方式を使用してください：
1. 「🔗 共有」で状態コードを取得（例: `SS|9613.2|-0.500|0.300|17|Earth`）
2. 「📥 読込」にコードを貼り付けて「復元」

### BGM が鳴らない
ブラウザのオートプレイ制限のため、BGMボタンをタップして明示的に有効化してください。

### 動作が重い
- 軌跡・小惑星帯・ラベルをOFFにする
- CME・探査機・L点をOFFにする
- 統一比率モードでの惑星フォーカスは高負荷
