# 🌌 太陽系シミュレーター

プロシージャル描画によるインタラクティブな太陽系・銀河系・惑星表面の3Dシミュレーター。外部画像アセット不使用、React + Canvas 2D APIのみで実装。

## 🚀 ライブデモ

[https://htanny.github.io/solar/](https://htanny.github.io/solar/)

## 📦 セットアップ

```bash
# クローン
git clone https://github.com/htanny/solar.git
cd solar

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build
```

開発サーバーは `http://localhost:5173` で起動します。

## 概要

太陽系の8惑星・太陽・月・ガリレオ衛星・彗星・小惑星帯、天の川銀河の3D構造、全8惑星の表面風景を、プロシージャルに描画するシミュレーター。

- **外部画像なし** — Canvas 2Dによる完全プロシージャル描画
- **物理計算** — ケプラー軌道、自転、地軸傾斜、太陽日、位相角影
- **3階層スケール** — 太陽系 / 恒星間 / 銀河系
- **着陸ビュー** — 各惑星の地表から周囲を見渡す一人称視点
- **時間旅行** — 任意の日付にジャンプして惑星配置を再現

## クイックスタート

1. シミュレーターを起動すると、**今日の日付の惑星配置**が表示されます
2. 画面をドラッグして視点を回転
3. ピンチ/スクロールでズーム
4. 惑星をタップで情報パネル表示
5. 左下パネルの「フォーカス」から対象選択

### 天体

| 天体 | 内容 |
|---|---|
| 太陽 | プロシージャル太陽テクスチャ（黒点12個・粒状斑20・白斑8・コロナ光線12本） |
| 8惑星 | 水星・金星・地球・火星・木星・土星・天王星・海王星 |
| 月 | 地球の衛星、公転周期27.3日、位相影 |
| ガリレオ衛星 | イオ・エウロパ・ガニメデ・カリスト（実軌道周期） |
| 彗星 | ハレー彗星（75年周期）・エンケ彗星（3.3年周期）、ケプラー方程式 |
| 小惑星帯 | 火星-木星間に200個 |
| 銀河系 | 天の川（渦巻4腕・バルジ・ダスト帯・近傍恒星85個） |

### ビジュアル

- 全惑星プロシージャルテクスチャ（地球の7大陸、木星の大赤斑、土星リング等）
- 位相角ベースの昼夜影（月の満ち欠けと同じ原理）
- 背景1400個の恒星（OBAF分類の8色、まばたき効果）
- 10個の星雲
- 軌道線・軌跡（フェード）・地軸表示

### モード

- **太陽系ビュー** - 標準モード
- **銀河系ビュー** - ズームアウトで天の川銀河を3D回転視点で観察
- **惑星表面ビュー** - 各惑星の地表から周囲を見渡す
- **サイズ比較モード** - 太陽・全惑星を実スケールで並列表示
- **ツアーモード** - 自動巡回（太陽→各惑星→彗星、4秒ホールド）

## ビューモード

### 太陽系ビュー（ズーム 0.04x〜150000x）

- 個別/統一の惑星半径・太陽半径・距離比率を切替可能
- 軌跡・軌道線・小惑星帯・月・ラベル・地軸の表示ON/OFF
- Saturnのリング、Jupiterのガリレオ衛星を表示

### 銀河系ビュー（ズーム 0.00002x〜0.03x）

- ズームアウトで太陽系がフェードアウト、天の川銀河がフェードイン
- **Gキーまたは −ボタン連打** で切替
- ドラッグで銀河を3D回転
- 名前付き恒星5個（シリウス・プロキオン・ベガ・アルタイル・デネブ）
- 中心バルジにラベル「銀河中心 ▸ いて座A*」
- スケールバー（1万光年）

### 惑星表面ビュー（着陸モード）

各惑星で異なる地表風景：

| 惑星 | 特徴 |
|---|---|
| 水星 | クレーター地形・黒い空・星空・巨大な太陽（6.7倍） |
| 金星 | 黄褐色ヘイズ・火山岩・稲妻フラッシュ・溶岩の赤い発光 |
| 地球 | 青空↔星空の昼夜遷移・流動する雲・草・木々・高緯度オーロラ |
| 火星 | 赤い空・砂塵パーティクル・ダストデビル・メサ |
| 木星 | 雲の海・縞模様・大赤斑（回転アニメーション）・雷・高緯度オーロラ |
| 土星 | 空を横切る壮大なリング（3本・影付き） |
| 天王星 | メタン氷結晶・淡い水色・極めて小さい太陽 |
| 海王星 | 濃紺の空・深い青の氷 |

#### 他の天体が見える
- 地球 → 月（位相影）、金星（明けの明星）
- 火星 → フォボス、デイモス、地球（青い点）
- 木星 → ガリレオ衛星4個（名前ラベル）
- 土星 → タイタン

#### 3層パララックス
- 遠景山脈 / 中景山脈 / 近景地形 が異なる速度で動く
- 奥行き感を強調

#### 環境音
- 地球: 風音＋鳥の鳴き声
- 火星: 薄い風音
- 金星: 深い圧力ドローン＋ヒス音
- 木星/土星: 嵐ランブル＋間欠的雷鳴
- 天王星/海王星: 唸る風
- 水星: 静寂＋熱膨張のピング音

#### HUD情報
- 惑星名・特徴・重力・太陽日・時間帯
- **緯度・経度・方位・太陽高度**
- FOV倍率（望遠鏡使用時）

## 物理モデル

### 軌道力学
- 惑星: 円軌道近似
- 彗星: **ケプラー方程式** `E = M + e·sin(E)` をニュートン法で解法（6反復）
- 真近点角: `v = 2·atan2(√(1+e)·sin(E/2), √(1-e)·cos(E/2))`

### 自転
- 太陽日計算: 順行時 `1/(1/rotAbs − 1/p)`、逆行時 `1/(1/rotAbs + 1/p)`
- 金星（243日逆行）→ 太陽日117日で西から昇り東へ
- 天王星（97.8°傾斜）→ 地軸が公転面にほぼ水平

### 影（昼夜）
- 3D空間で太陽方向ベクトルを計算
- 視点変換後、位相角に基づいてターミネーターを楕円描画
- 月の満ち欠けと同じ原理

### 太陽高度（着陸ビュー）
```
sin(高度) = sin(緯度)·sin(赤緯) + cos(緯度)·cos(赤緯)·sin(時角)
```
- 赤緯: `effTilt × sin(t/p × 2π)` （季節変動）
- 有効傾斜: 90°以上の惑星は `180−t` で扱う

### 検証例（赤道での夏至正午の太陽高度）

| 惑星 | 計算値 | 理論値 |
|---|---|---|
| 水星 | 90.0° | 90°（傾斜0°） |
| 地球 | 66.6° | 90°−23.4°=66.6° ✓ |
| 火星 | 64.8° | 90°−25.2°=64.8° ✓ |
| 天王星 | 7.8° | 90°−82.2°=7.8° ✓ |

## 技術仕様

### アーキテクチャ

```
solar-system.jsx (単一ファイル、約980行)
├── データ定義
│   ├── PL（8惑星）
│   ├── COMETS（2彗星）
│   ├── GMOONS（ガリレオ衛星）
│   ├── SURF（地表データ）
│   └── PL_MAP/COMET_MAP（O(1)参照）
├── 描画関数
│   ├── drawPlanetBody（惑星テクスチャ）
│   ├── drawSun（太陽・黒点・コロナ）
│   ├── drawLanding（地表ビュー）
│   └── ユーティリティ（fillCirc・dSh・dRi等）
├── 3Dユーティリティ
│   ├── RX/RY（軸回転）
│   ├── pj（3D→2D投影）
│   └── galPj（銀河投影）
├── Reactコンポーネント
│   ├── DragPanel（ドラッグ可能UIパネル）
│   └── App（メインコンポーネント）
└── 音響
    ├── BGM（Web Audio アンビエント）
    └── startLandSound/stopLandSound（環境音）
```

### 状態管理
- **useState**: UI状態（ボタン・モード切替）
- **useRef**: アニメーションループ用状態（再レンダリング回避）
- 各state変化時にrefを同期（`useEffect`経由）

### パフォーマンス最適化
- `PL_MAP`/`COMET_MAP`: O(1)ルックアップ
- アニメーションフレームからの`setState`は最小限
- スター描画: 画面外カリング
- ズームに応じた描画詳細度の動的調整

### Artifact環境制約への対応
| 制限 | 対応 |
|---|---|
| `canvas.toDataURL` ブロック | クリーンビューモード（UI非表示→ネイティブスクリーンショット） |
| `navigator.clipboard` ブロック | `execCommand("copy")` フォールバック |
| URL共有不可 | テキスト状態コード（`SS|t|rx|ry|z|foc`形式） |
| `localStorage` 未使用 | すべてメモリ内 |

## トラブルシューティング

### 📷 撮影モードで画像が保存できない
Artifact環境では `canvas.toDataURL` がブロックされているため、撮影モードは**UIパネル非表示状態**にするだけです。

**保存方法:**
- Android: 電源ボタン＋音量↓ を同時押し
- iOS: サイドボタン＋音量↑ を同時押し

### 🔗 共有URLが機能しない
Artifact環境ではURLが一時的なため、URLリンクは機能しません。**状態コード方式**を使用してください：

1. 「🔗 共有」で状態コードを取得（例: `SS|9613.2|-0.500|0.300|17|Earth`）
2. 相手にコードをLINE/メール等で送信
3. 相手は「📥 読込」にコードを貼り付けて「復元」

### BGM が鳴らない
ブラウザのオートプレイ制限により、**BGMボタンをタップして明示的に有効化**する必要があります。

### ランディング中に他惑星が選択された
フォーカスを変更すると、自動的に新しい惑星へ着陸します。離陸する場合は🚀離陸ボタンか`Esc`キー。

### 動作が重い
- モバイルデバイスでは星の数を減らす、軌跡をOFFにすると改善
- 小惑星帯・ラベルをOFFにする
- サイズ比較モード/ツアー中は描画負荷が低い
